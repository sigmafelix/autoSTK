# Autodetect the temporal unit in a lubridate/POSIXct object

Autodetect the temporal unit in a lubridate/POSIXct object

## Usage

``` r
detect_temporal_unit.generic(temporal)
```

## Arguments

- temporal:

  a lubridate/POSIXct object.

## Value

A character that indicates the temporal unit of the input xts object.
